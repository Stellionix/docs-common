# Troubleshooting

## Common Problems

!!! warning "Plugin does not start"
    Verify server version, Java version and dependency conflicts first.

!!! note "Chest not generated"
    Check generation rules, excluded worlds, permissions and integration toggles.

!!! info "Command missing"
    Confirm the plugin enabled correctly and that the user has the expected permission.

## Quick checklist

- Review startup logs
- Confirm config syntax
- Test with a minimal plugin stack
- Reproduce on a clean world if necessary
