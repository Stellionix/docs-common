# Commands and Permissions

## Core Commands

| Command | Description | Permission |
| --- | --- | --- |
| `/dc list` | List active chests | `deadchest.list.own` |
| `/dc reload` | Reload configuration | `deadchest.admin` |
| `/dc removeall` | Remove all active chests | `deadchest.admin` |
| `/dc ignore` | Edit ignored items | `deadchest.admin` |

## Operational Notes

- Keep command tables compact and scannable.
- Inline code density should stay readable in both color schemes.
- This page intentionally mirrors a common admin-doc format.
