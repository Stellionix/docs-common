# Customization

## Customization Surface

Use this page to verify how the design behaves with longer admin-oriented copy.

### Typical customization areas

- Message wording
- Container block type
- Lifetime and cleanup rules
- Filters and ignored items
- Particle and sound feedback

### Example snippet

```yaml
chest:
  block-type: barrel
  duration-seconds: 900

visuals:
  effect-animation:
    enabled: true
    style: soul
```

!!! info "Design intent"
    Customization pages are useful for testing the balance between prose,
    code blocks and secondary headings.
